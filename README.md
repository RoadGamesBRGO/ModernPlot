# ModernPlot
# 🗺️ ModernPlot — Proteção de Terrenos Simples e Eficaz

> O sistema definitivo para seus jogadores protegerem suas construções contra griefers e invasores.

---

## 📖 Sobre o ModernPlot
O **ModernPlot** permite que os jogadores comprem, protejam e gerenciem seus próprios terrenos (plots) de forma automatizada. Ele é integrado ao sistema de economia e garante que cada centímetro do seu mundo tenha um dono legítimo.

---

## 🛠️ Comandos de Terrenos

<details>
<summary><b>🏡 Comandos do Jogador (Clique para abrir)</b></summary>

*   **/plot claim** - Compra e protege o terreno onde você está pisando.
*   **/plot info** - Mostra quem é o dono e quem tem permissão no terreno.
*   **/plot add <jogador>** - Dá permissão para um amigo construir no seu plot.
*   **/plot remove <jogador>** - Retira a permissão de um jogador.
*   **/plot delete** - Exclui o seu terreno (cuidado: não há reembolso!).
*   **/plot list** - Lista todos os seus terrenos protegidos.
</details>

<details>
<summary><b>👮 Comandos de Staff (Clique para abrir)</b></summary>

*   **/plot admin delete** - Remove qualquer terreno (mesmo de outros jogadores).
*   **/plot admin info** - Mostra detalhes avançados de proteção.
*   **/plot bypass** - Permite ao administrador construir em qualquer lugar (modo deus).
</details>

<details>
<summary><b>🔐 Permissões</b></summary>

| Permissão | Função |
| :--- | :--- |
| `modernplot.command.user` | Permite usar /plot claim, info e list |
| `modernplot.command.manage` | Permite gerenciar amigos (/plot add/remove) |
| `modernplot.admin.all` | Permite gerenciar qualquer terreno do servidor |

</details>

---

## 🚀 Como Configurar
1. Instale o **ModernEconomy** (obrigatório para o sistema de compra).
2. Coloque o **ModernPlot.phar** na pasta `/plugins/`.
3. No arquivo `config.yml`, defina o **preço por bloco** e o **tamanho máximo** de cada terreno.
4. Reinicie o servidor e pronto!
